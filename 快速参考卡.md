## ⚡ 快速故障排查

| 问题 | 解决 |
|------|------|
| 启动脚本无反应 | 检查 Python: `python --version` |
| 依赖缺失 | `pip install -r requirements.txt` |
| 8501 被占用 | `netstat -ano \| findstr 8501` |
| GitHub 同步失败 | 检查网络 / 运行 `git push` |
