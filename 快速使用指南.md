# 📖 快速使用指南 - 执剑人系统

## 🚀 一键启动应用

### 方式 1：双击启动（推荐）
直接双击 **`一键启动.bat`** 文件

✨ 系统会自动：
- ✅ 检查 Python 环境
- ✅ 检查依赖是否安装（首次会自动安装）
- ✅ 启动 Streamlit 应用
- ✅ 自动打开浏览器

### 方式 2：添加快捷方式到桌面
1. 右键点击 **`一键启动.bat`**
2. 选择 **"发送到"** → **"桌面（创建快捷方式）"**
3. 之后在桌面双击快捷方式即可启动

### 方式 3：固定到任务栏（Windows 10/11）
1. 右键点击桌面快捷方式
2. 选择 **"固定到任务栏"**
3. 之后直接在任务栏点击即可启动

## 📱 应用访问方式

### 本地访问
启动后自动打开浏览器，或手动访问：
```
http://localhost:8501
```

### 手机/平板访问（同一网络）
启动后会显示以下地址之一：
```
http://你的电脑IP:8501
或
http://电脑名称.local:8501
```

**查询你的 IP 地址：**
```powershell
ipconfig
```
找 IPv4 Address（如：192.168.1.100）

## 🛠️ 故障排查

### 问题：启动脚本没反应

**解决方案：**
```powershell
# 1. 进入项目目录
cd "e:\Wallfacer System"

# 2. 直接运行（查看错误详情）
streamlit run app_v2.py

# 3. 如果有错误，根据错误信息解决
```

### 问题：8501 端口被占用

**解决方案：**
```powershell
# 1. 查看占用进程
netstat -ano | findstr 8501

# 2. 杀死进程（假设 PID 是 12345）
taskkill /PID 12345 /F

# 3. 重新运行脚本
```

### 问题：依赖缺失

**解决方案：**
```powershell
cd "e:\Wallfacer System"
pip install -r requirements.txt
```

### 问题：GitHub 同步失败

**解决方案：**
```powershell
# 1. 检查网络连接
# 2. 检查 Token 是否过期
# 3. 手动同步：
git add .
git commit -m "描述"
git push
```

## 📊 数据管理

### 查看保存的数据
```powershell
# 数据保存在:
# e:\Wallfacer System\wallfacer_data.db

# 导出数据到 CSV:
# 在应用中点击 "数据面板" → "所有计划" → "导出 CSV"
```

### 备份数据库
```powershell
# 复制数据库文件到安全位置
Copy-Item "wallfacer_data.db" "C:\Backup\wallfacer_data_backup.db"
```

### 清空数据（谨慎！）
```powershell
# 删除数据库，重新启动应用会创建新的
Remove-Item "wallfacer_data.db"
```
